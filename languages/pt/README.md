## O Que é Este Recurso?

Este é o repositório mais completo de **casos de uso de Inteligência Artificial** projetados especificamente para o **Caribe e América Latina**. Cada caso de uso é:

- **Específico por país**: Adaptado às realidades econômicas, indústrias e desafios de cada nação
- **Prático**: Implementações do mundo real com prompts prontos para usar
- **Acessível**: Projetado para pequenas empresas, empreendedores e profissionais
- **Multilíngue**: Disponível em inglês, espanhol, português, francês e crioulo haitiano

## 10 Casos de Uso Essenciais para PMEs Brasileiras

### 1. Marketing para Redes Sociais
```
Sou dono de um negócio de [tipo de negócio] em [cidade, estado].
Meus produtos/serviços: [descrição breve].
Meus clientes-alvo: [descrição].
Personalidade da marca: [divertida/profissional/acolhedora/arrojada].
Tema/temporada do mês: [descreva].

Crie para mim:
- 8 posts para Instagram (mix: promocional, educativo, engajamento)
- 4 posts para Facebook
- 4 hooks para TikTok/Reels
- 2 mensagens para lista de transmissão WhatsApp
- 10 hashtags relevantes para o mercado brasileiro

Escreva em português brasileiro, com linguagem natural e autêntica.
```

### 2. Plano de Negócio para MEI/ME
```
Ajude-me a criar um plano de negócios para [tipo de negócio] no Brasil.
Estrutura: [MEI/ME/EPP].
Investimento disponível: [valor em BRL].
Mercado-alvo: [descrição].

Inclua: análise de mercado específica para [estado/cidade],
projeções financeiras realistas para o contexto brasileiro,
regime tributário recomendado (Simples Nacional/Lucro Presumido),
análise da concorrência local,
estratégia de marketing digital,
e plano de ação para os primeiros 90 dias.
```

### 3. Atendimento via WhatsApp Business
```
Crie um sistema de respostas automáticas para o WhatsApp Business
do meu [tipo de negócio] no Brasil.
As 10 perguntas mais frequentes dos meus clientes:
1. [pergunta 1]
2. [pergunta 2]
[continuar...]

Escreva respostas claras, simpáticas e objetivas (máx. 100 palavras),
em português brasileiro natural.
Inclua emojis onde apropriado para o tom da marca.
```

### 4. Gestão de Pix e Cobranças
```
Crie templates de comunicação para cobranças e confirmações de pagamento
para meu negócio no Brasil.
Tipo de negócio: [descrição]. Ticket médio: [valor BRL].
Meios de pagamento aceitos: [Pix/cartão/boleto].

Crie:
- Mensagem de confirmação de pagamento via Pix (WhatsApp)
- Lembrete de cobrança amigável (1 dia antes)
- Cobrança firme para inadimplentes (7 dias)
- Mensagem para cliente com pagamento em dia (fidelização)
- Template de nota fiscal simplificada (MEI)
```

### 5. Conteúdo para SEO Brasileiro
```
Escreva um artigo de blog de [X palavras] para [website]
sobre [tema relevante para o mercado brasileiro].
Palavra-chave alvo: [palavra-chave em português].
Público: [descrição].
Estado/região de foco: [estado/região do Brasil].

Inclua: dados do mercado brasileiro, exemplos locais,
referências a legislação brasileira quando relevante,
e otimização para SEO em português.
```

### 6. Declaração de IR Simplificada (Orientação)
```
Oriente-me sobre a declaração do Imposto de Renda para [tipo de situação]
no Brasil para o ano-base [ano].
Situação: [empregado/autônomo/MEI/pequeno empresário].
Renda aproximada: [faixa].
Bens e direitos: [breve descrição].
Dívidas e ônus: [se houver].

Explique: obrigatoriedade de declaração, prazo,
documentos necessários, principais deduções permitidas,
diferença entre declaração simplificada e completa,
e como usar o programa da Receita Federal.
Nota: oriente a confirmar com contador.
```

### 7. LGPD — Política de Privacidade
```
Crie uma política de privacidade simples para meu [tipo de negócio]
no Brasil, em conformidade com a LGPD (Lei 13.709/2018).
Dados coletados: [lista: nome, email, CPF, etc.].
Finalidade da coleta: [para que usa os dados].
Plataformas usadas: [site/app/WhatsApp/redes sociais].
Compartilhamento de dados: [com quem compartilha, se houver].

Crie: política de privacidade (linguagem acessível),
aviso de cookies para site,
formulário de consentimento,
e processo de solicitação de exclusão de dados.
```

### 8. E-commerce no Brasil
```
Crie uma estratégia de e-commerce para um negócio no Brasil vendendo [produtos].
Plataforma atual: [Mercado Livre/Shopee/Magalu/site próprio/nenhuma].
Ticket médio: [valor BRL]. Volume mensal: [X pedidos].
Logística: [Correios/Jadlog/própria/Fulfillment].
Região de entrega: [cidade/estado/todo Brasil].

Recomende: melhores plataformas para [tipo de produto],
integração com Pix e parcelamento,
estratégia de frete (frete grátis threshold),
gestão de devoluções (Código de Defesa do Consumidor),
e como competir com grandes players no Brasil.
```

### 9. Captação de Recursos (Editais e Programas)
```
Pesquise oportunidades de financiamento para [tipo de negócio]
no Brasil para [finalidade: expansão/inovação/capital de giro].
Porte: [MEI/ME/EPP]. Setor: [setor econômico].
Estado: [estado].
Valor necessário: [faixa em BRL].

Liste: programas do BNDES disponíveis,
linhas do Banco do Brasil e Caixa para PMEs,
editais abertos do SEBRAE,
programas estaduais em [estado],
e fundos de inovação (FINEP, FAPESP, FAPERJ etc.),
com critérios de elegibilidade e prazo.
```

### 10. Análise de Concorrência
```
Analise a concorrência para [tipo de negócio] em [cidade/estado, Brasil].
Meus concorrentes principais: [lista].
Meus pontos fortes: [lista].
Minhas fraquezas: [lista].
Contexto de mercado: [crescimento/estabilidade/retração].

Crie: perfil de cada concorrente (forças/fraquezas),
análise de preços praticados,
oportunidades de diferenciação,
e 3 recomendações estratégicas baseadas no contexto brasileiro.
```
