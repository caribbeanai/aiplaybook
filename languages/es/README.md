## ¿Qué es este recurso?

Este es el repositorio más completo de **casos de uso de Inteligencia Artificial** diseñados específicamente para el **Caribe y América Latina**. Cada caso de uso es:

- **Específico por país**: Adaptado a las realidades económicas, industrias y desafíos de cada nación
- **Práctico**: Implementaciones del mundo real con prompts listos para usar
- **Accesible**: Diseñado para pequeñas empresas, emprendedores, funcionarios públicos y profesionales
- **Multilingüe**: Disponible en inglés, español, portugués, francés y criollo haitiano

## 10 Casos de Uso de IA Esenciales para PyMEs de América Latina

### 1. Marketing en Redes Sociales
```
Soy dueño de un negocio de [tipo de negocio] en [ciudad, país].
Mis productos/servicios: [descripción breve].
Mis clientes objetivo: [descripción].
Personalidad de marca: [divertida/profesional/cálida/audaz].
Tema/temporada del mes: [describe].

Crea para mí:
- 8 publicaciones para Instagram (mix: promocional, educativo, interactivo)
- 4 publicaciones para Facebook
- 4 hooks para TikTok/Reels
- 2 mensajes de difusión para WhatsApp Business
- 10 hashtags relevantes para mi mercado en [país]
```

### 2. Plan de Negocio
```
Ayúdame a crear un plan de negocios para [tipo de negocio] en [país].
Inversión disponible: [monto en moneda local].
Mercado objetivo: [descripción].
Experiencia: [años de experiencia en el sector].

Incluye: análisis de mercado específico para [país],
proyecciones financieras a 3 años,
análisis de competencia local,
estrategia de marketing digital,
y un plan de acción para los primeros 90 días.
```

### 3. Atención al Cliente por WhatsApp
```
Crea un sistema de respuestas automáticas para WhatsApp Business
para mi [tipo de negocio] en [país].
Las 10 preguntas más frecuentes de mis clientes:
1. [pregunta 1]
2. [pregunta 2]
[continuar...]

Escribe respuestas claras, amigables y breves (máx. 100 palabras cada una),
en español apropiado para [país].
```

### 4. Estrategia de Precios
```
Ayúdame a establecer precios para [producto/servicio] en [país].
Mis costos: [desglose].
Precio de la competencia: [lista].
Mi diferenciación: [qué me hace especial].
Moneda: [moneda local].

Recomienda: modelo de precios, puntos de precio específicos,
cómo justificar el precio, política de descuentos
y cómo aumentar precios sin perder clientes.
```

### 5. Contenido para Blog/SEO
```
Escribe un artículo de blog de [X palabras] para [website] sobre [tema].
Keyword objetivo: [palabra clave en español].
Audiencia: [descripción].
País/región de enfoque: [país/región de América Latina].
Incluye: estadísticas del mercado local, ejemplos locales,
y optimización SEO para búsquedas en español.
```

### 6. Análisis de Competencia
```
Analiza mi competencia para [tipo de negocio] en [ciudad, país].
Mis competidores principales: [lista].
Mis fortalezas actuales: [lista].
Mis debilidades: [lista].

Crea: perfil de cada competidor, mapa de posicionamiento,
oportunidades de mercado desaprovechadas,
y 3 recomendaciones estratégicas para diferenciarme.
```

### 7. Capacitación de Personal
```
Crea materiales de capacitación para [rol] en [tipo de empresa] en [país].
Proceso a capacitar: [descripción].
Nivel educativo del personal: [básico/intermedio/profesional].
Idioma: español para [país] (considera regionalismos).

Incluye: manual de procedimientos, lista de verificación,
escenarios de rol para práctica,
y evaluación de conocimientos.
```

### 8. Propuesta Comercial
```
Escribe una propuesta comercial para [proyecto/servicio]
para [tipo de cliente] en [país].
Alcance del proyecto: [descripción detallada].
Precio: [monto en moneda local].
Mis calificaciones: [experiencia relevante].
Plazo: [duración].

Formato profesional con: resumen ejecutivo,
descripción del problema, solución propuesta,
entregables, cronograma, inversión, y términos.
```

### 9. Gestión de Crisis de Reputación
```
Mi empresa en [país] está enfrentando una crisis por: [descripción del problema].
Plataformas afectadas: [redes sociales, prensa, WhatsApp].
Estado actual: [describe qué se está diciendo].

Crea: comunicado oficial (tono apropiado para [país]),
respuestas para redes sociales,
protocolo de comunicación interna,
y plan de recuperación de reputación a 30 días.
```

### 10. Solicitud de Crédito / Financiamiento
```
Ayúdame a preparar una solicitud de financiamiento para
[banco/programa de gobierno/fondo de desarrollo] en [país].
Mi negocio: [nombre, tipo, años de operación].
Uso del financiamiento: [propósito específico].
Monto solicitado: [cantidad en moneda local].
Impacto: [empleos creados, crecimiento de ventas esperado].

Crea: resumen ejecutivo (300 palabras), descripción del negocio,
plan de uso de fondos, proyección financiera,
y carta de presentación.
```

- [Ver Guía Completa de Turismo](../../sectors/tourism-and-hospitality.md)

## Riesgos de IA que Debes Conocer

Antes de implementar IA en tu empresa o institución, lee nuestra
[Guía Completa de Riesgos de IA](../../risks/README.md) que cubre:

- Privacidad de datos (LGPD Brasil, Ley 1581 Colombia, LFPDPPP México)
- Sesgos algorítmicos en contexto latinoamericano
- Desplazamiento laboral y estrategias de adaptación
- Desinformación y deepfakes antes de elecciones
- Soberanía digital y dependencia tecnológica
