## What Are Claude Skills?

**Claude Skills** are installable AI capability packages built to the official Anthropic Skills
standard. Each skill is a self-contained folder containing a `SKILL.md` file with structured
instructions, regional context, workflows, and examples — packaged so that Claude instantly
becomes a specialist for that domain.

This collection provides **Claude Skills purpose-built for the Caribbean and Latin American
business context** — not generic global templates, but skills that understand island economies,
regional trade frameworks, local currencies, cultural marketing, and the real challenges faced
by Caribbean and LATAM entrepreneurs.

### How to Install a Skill

1. Download the skill folder (e.g., `caribbean-marketer/`)
2. In Claude.ai: Go to **Settings → Skills → Add Skill** and upload the folder
3. In Claude Code: Place the folder in your `~/.claude/skills/` directory
4. Claude will automatically detect when to activate each skill based on your conversation

## Skill Architecture

Each skill follows the official **Anthropic Skills Standard**:

```
skill-name/
├── SKILL.md          ← Core skill file (YAML frontmatter + instructions)
└── references/       ← Supplementary documentation (loaded as needed)
    └── *.md
```

**Three-level progressive disclosure**:
1. **YAML frontmatter**: Always in Claude's system prompt — tells Claude when to activate
2. **SKILL.md body**: Loaded when the skill is relevant — full instructions, frameworks, examples
3. **references/ files**: Linked documentation discovered as needed — detailed guides, country data
