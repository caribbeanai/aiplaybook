## What Are Claude Skills?

**Claude Skills** are pre-built, reusable AI capability frameworks that you can use directly with Claude (claude.ai) or adapt for other AI tools. Think of them as professional-grade AI tools packaged for everyday use.

Each skill includes:
- **What it does**: Clear description
- **Who it's for**: Target role or business type
- **Ready-to-use prompt**: Copy, customize, use
- **Expected output**: What you'll get back
- **Tip**: How to get the best results
